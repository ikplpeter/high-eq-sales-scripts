# HighEQ Sales Scripts
## 高情商销售与沟通话术工具

---

### 📖 项目简介
这是一套专为 OpenClaw 打造的**高情商沟通与销售话术技能**，覆盖日常生活、职场博弈、全行业门店会员卡营销、洗浴/水疗会所增值服务推销、销售边界防护与防骚扰避险。

核心心法：**不落入对方框架、认知重构定义、温柔有边界、气场不撕破脸、先判场景身份再输出话术**。

## Version Note · 1.0.0
This release V1.0.0 is the **initial foundational version** of HighEQ Sales Scripts.

It prioritizes two core scenarios as the first official baseline:
1. Spa / wellness club membership sales, service value packaging, customer intent screening and graceful exit strategies.
2. Daily social & workplace communication: high-EQ response, verbal de-escalation, boundary setting and workplace anti-bullying tactics.

This version locks the core psychological logic and standard script library.
Future updates will expand to more industries, business negotiation and social scenarios.

### ✨ 核心功能
1. **生活/职场高情商化解**
应对恶意玩笑、负面标签、领导调侃、下属抬杠、同事阴阳怪气，做到不卑不亢、体面化解、气场立界。
2. **全行业会员卡营销话术**
适配餐厅、理发店、实体商号，通过痛点唤醒、场景绑定、身份特权塑造，自然创造办卡充值需求。
3. **洗浴/水疗会所专属营销体系**
专属客服、车接车送增值服务高阶包装话术，主打省心、隐私、排面、高端熟客专属权益。
4. **客户意愿甄别与体面退场**
三步快速判断办卡意向，无意向客户用自然借口优雅抽身，不纠缠、不耗时间、不伤和气。
5. **销售防骚扰与刑事风险规避**
建立职业人设与公私边界，应对半夜邀约、暧昧调侃、肢体试探、私礼馈赠，保护自身安全与职业底线。

### 🌐 English Overview
A comprehensive OpenClaw skill for high-EQ communication and sales. It provides tactful scripts for resolving awkward situations, upselling memberships in hospitality, setting professional boundaries, and preventing harassment.

Key features include cognitive reframing, assertive communication, and graceful exit strategies for low-intent customers.
