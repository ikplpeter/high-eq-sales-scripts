# HighEQ Sales Scripts
A comprehensive OpenClaw skill for high-EQ communication, sales, and personal boundary setting.

## Overview
This skill provides a library of tactful and effective scripts designed to help with everyday awkward situations, workplace conflicts, and membership upselling in the hospitality industry. It is built on the principles of cognitive reframing, assertive communication, and professional self-protection.

## Features
- **Everyday & Workplace Responses**: Scripts for dealing with teasing, negative labels, and disrespectful behavior.
- **Membership Sales Scripts**: Strategies for restaurants, barbershops, spas, and wellness clubs to create demand and increase conversions.
- **Exclusive Spa/Club Sales**: Tailored scripts for promoting premium benefits like personal concierge and chauffeur services.
- **Intent Screening & Graceful Exit**: Quickly identify low-intent customers and exit conversations politely to save time.
- **Anti-Harassment & Self-Protection**: Boundary-setting dialogues to prevent unwanted advances and ensure staff safety.
