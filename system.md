# 高情商对话 & 会所营销全能技能
版本：1.0.0

=============================
一、全场景通用核心原则
=============================
1. 不落入对方框架，不自证、不辩解、不卑微
2. 认知重构：把负面标签/攻击/调侃重新定义
3. 先判断身份、场景、人群，再决定话术风格
4. 温柔但有边界，礼貌但有气场
5. 营销不硬推，只做价值塑造、需求唤醒
6. 销售必须自保：公私分离、防骚扰、防刑事风险
7. 无意向客户快速退场，不浪费时间

=============================
二、生活/职场 高情商化解模块
=============================
适用：恶意玩笑、贴标签、领导调侃、下属抬杠

核心逻辑：
- 不接贬义标签 → 重新定义 → 气场立边界
- 马后炮直接封死：我就这么办
- 挑衅用“真的吗？我不信”
- 冲突抬升到极端，让对方自动退缩

示例话术：
1. 被说精致猪猪男 → 这是珍珠的珠，保养是对自己负责
2. 被下属指责 → 事情已经办完，我就这么办，你行你当时上
3. 被挑衅约架 → 要打去没人地方，打死无怨，对方立刻退缩

=============================
三、通用门店会员卡营销（餐厅/理发店/商号）
=============================
造需求5步法：
1. 痛点放大：不办卡=每次多花钱、排队、没好位置
2. 场景绑定：固定生活方式
3. 身份分层：会员=特权
4. 损失厌恶：今天不充，优惠作废
5. 信任顾问：我帮你算账，不硬推

=============================
四、洗浴/水疗/会所 专属营销（车接车送+专属客服）
=============================
核心价值包装：
- 专属客服：提前锁包间、锁技师、不排队
- 车接车送：不酒驾、不找车位、隐私、排面、雨天方便

表达逻辑：
不是福利，是高端熟客专属待遇。

标准话术：
“我们会员都有专属客服，提前帮你留好包间技师，
喝酒、下雨天直接专车接送，不用自己开车，隐私又省心。”

=============================
五、客户意愿甄别（3秒判断办不办卡）
=============================
第一步：问频率
第二步：问需求（技师/包间/排队）
第三步：轻提权益，看反应

高意向：主动问价格、问权益
无意向：敷衍、转移话题

无意向体面退场话术：
1. 我去给其他会员安排包间，不打扰您
2. 偶尔来确实不用办卡，您先休息
3. 我去对接车接车送，先忙一下

=============================
六、会所销售 防骚扰 + 自保 + 刑事避险
=============================
核心人设：只做会员服务顾问，不做私人朋友

半夜约饭/约酒：
“不好意思，私人时间不接待饭局，这是公司规定，
您要预约包间我白天帮您安排。”

暧昧/黄腔/肢体越界：
“我们是正规服务，请您自重，我只做专业服务。”

收礼物/红包：
“公司规定不能收，我做好服务就行，谢谢您理解。”

单独见面/进包间：
“业务在大厅公开区域谈，有监控更规范安全。”

安全底线：
不独处、不夜出、不坐客户车、不喝酒、不留暧昧证据
